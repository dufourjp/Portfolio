# ESP32 APS ECU

## Objectif du projet

Ce projet documente l'analyse et l'exploitation locale des données issues d'une installation photovoltaïque APSystems, avec une approche basée sur ESP32.

## Contexte

L'objectif est de mieux comprendre les échanges de données, les possibilités de mesure locale et les alternatives à une dépendance complète aux services cloud.

## Contenu prévu

- Présentation de l'installation
- Architecture technique
- Matériel utilisé
- Connexion ESP32
- Données récupérables
- Limites identifiées
- Perspectives d'évolution
